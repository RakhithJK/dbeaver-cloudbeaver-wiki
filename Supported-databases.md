### Supported databases:
* ClickHouse
* DB2 iSeries/AS 400
* DB2 LUW
* Derby Server
* Firebird
* H2 Embedded
* MariaDB
* MySQL
* Oracle
* PostgreSQL
* SQL Server
* SQLite
* Trino

**Note**: you can add other drivers manually using [the instruction](https://github.com/dbeaver/cloudbeaver/wiki/Adding-new-database-drivers).
